## 3. 리포트 내용

대상: 선택한 연·월(예: 2026-05)에 **마감(period_end)이 속하는 정산서 전체**.

### 3.1 파일 형식
- **CSV (UTF-8 with BOM)** — 한글 Excel에서 깨짐 없이 바로 열림. 신규 라이브러리 불필요.
- 파일명: `WhiteOn_정산리포트_2026-05.csv`

### 3.2 컬럼 (정산서 1건 = 1행)
```
지사명 | 정산기간 | 정산상태 | 건수 | 매출(VAT포함) | 공급가액 | 부가세 | 수수료율(%) | 본사수수료 | 환불차감 | 지사지급액
```
- 맨 아래 **합계 행** 1줄: 전 지사 매출·공급가·부가세·수수료·지급액 총합.
- 금액은 모두 원 단위 정수, 천단위 구분 없이(회계SW import 호환).

### 3.3 집계 규칙
- "그 달에 마감된 정산서" = `period_end`의 연·월이 선택한 연·월과 일치.
- 모든 금액은 정산서에 이미 저장된 값을 그대로 사용 (재계산하지 않음 — §4.2가 확정한 숫자 보존).
- 상태 무관 전체 포함하되 상태 컬럼으로 구분 (DRAFT~INVOICED·HOLD).

## 4. 구현 — §4.6a

### 4.1 백엔드
- `app/services/report.py` (신규) — `build_monthly_csv(session, year, month) -> str`
  - 해당 월 정산서 조회 → CSV 문자열 생성. Decimal 사용.
- `app/api/v1/endpoints/settlements.py`에 엔드포인트 추가:
  - `GET /settlements/report/monthly?year=&month=` (HQ_ADMIN 전용)
  - `Response`로 CSV 반환, `Content-Disposition: attachment` 헤더 + BOM.

### 4.2 프론트엔드 — `www/admin.html` 정산 탭
- 기존 정산 관리 탭 상단에 "월별 회계 리포트" 영역 추가:
  - 연·월 선택 + **[엑셀 다운로드]** 버튼 → 위 엔드포인트 호출 → 파일 다운로드.
- 기존 정산서 생성·목록 영역은 그대로 보존 (harness_rules.md).
